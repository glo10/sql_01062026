# Exercice 0.11.2 : les fonctions regex

## Enoncé

1. Affichez tous les produits en boîte[s].
2. Affichez uniquement les domaines des e-mails des clients (ce qui vient après le @).
3. Affichez les numéros de téléphone des clients en supprimant tous les caractères qui ne sont pas des chiffres (les tirets, espaces, parenthèses, etc.).
4. Affichez le nombre d’espace, tirets ou parenthèses présents dans les numéros de téléphones de fournisseurs.
